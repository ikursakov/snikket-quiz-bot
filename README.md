# Snikket Quiz Bot

XMPP quiz bot for personal use.

## Features
- Direct message quiz
- Statistics tracking
- Parent reporting

## Config

```bash
cp config.example.json config.json
```

And fill in your credentials.

## Setup

```bash
python3 -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
python bot.py
```
